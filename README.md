# 🛡️ LinkGuardian

**Ultimate Link Protection for Global Influencers.**

LinkGuardian is a premium SaaS designed to monitor social media bio links 24/7. Built on a serverless architecture, it ensures your audience never lands on a broken page, protecting your followers, reputation, and revenue.

## 🚀 Features

- **24/7 Global Monitoring**: Automated bots check your links every hour from multiple global locations.
- **Instant Alerts**: Receive real-time notifications via Resend (Email) as soon as a link failure is detected.
- **Global Payments**: Integrated with PayPal for seamless subscription management anywhere in the world.
- **Modern Dashboard**: Clean, minimalist UI designed for the next generation of creators.

## 🛠️ Technology Stack

- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router)
- **Styling**: Vanilla CSS with a custom-built Design System
- **Email**: [Resend](https://resend.com/)
- **Payments**: [PayPal Smart Buttons](https://developer.paypal.com/)
- **Deployment**: [Vercel](https://vercel.com/) (Serverless & Cron Jobs)

## 📦 Deployment

1. Push this repository to GitHub.
2. Import the project into **Vercel**.
3. Configure the following Environment Variables:
   - `RESEND_API_KEY`: Your Resend API key.
   - `NEXT_PUBLIC_PAYPAL_CLIENT_ID`: Your PayPal REST App Client ID.
   - `CRON_SECRET`: A secure random string for Cron authentication.
4. Connect your domain: `linkguardian.net`.
